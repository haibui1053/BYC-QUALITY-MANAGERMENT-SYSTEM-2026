    ⚙ GitHub Sync

    This app already points to a default shared GitHub repo — just opening it and refreshing shows everyone the latest data, no setup needed for viewing. Only fill in a token below if you need to add or edit issues from this device.

    Only viewing? Leave the token empty — for a public repo, anyone can read the shared data with just the repo name, no token needed.

    Need to add/edit issues? Create a fine-grained token (github.com → Settings → Developer settings → Fine-grained tokens) scoped to only this one repo with "Contents: Read and write" permission — not a token with full account access. The token is stored only in this browser's local storage and is never uploaded anywhere except to GitHub's own API. Don't share your token — if your boss also needs to add/edit issues, they should create their own token.
